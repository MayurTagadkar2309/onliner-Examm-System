# 📚 Online Examination System — ACS College, Sonai

> A secure, browser-based online examination portal for students and administrators at **Arts, Commerce & Science College, Sonai** (Savitribai Phule Pune University).

## ✨ Features

| Feature | Description |
|---|---|
| 🔐 **Secure Login** | Separate login portals for Students and Admins |
| 📝 **Online MCQ Exams** | Attempt multiple-choice exams with an auto-countdown timer |
| 📊 **Instant Results** | Results are displayed immediately after exam submission |
| 👥 **Admin Panel** | Manage students, create exams, add questions, and view results |
| 📋 **Student Registration** | New students can register and get access to assigned exams |
| 📱 **Responsive Design** | Works on desktop, tablet, and mobile devices |

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- No server or build tools required — this is a **pure frontend** project

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/online-exam-system.git
   ```

2. **Navigate into the project folder**
   ```bash
   cd online-exam-system
   ```

3. **Open `index.html` in your browser**
   ```bash
   # Option 1: Just double-click index.html
   # Option 2: Use VS Code Live Server extension
   # Option 3: Use a simple HTTP server
   npx serve .
   ```

## 📸 Pages Overview

### 🏠 Home Page (`index.html`)
The landing page that introduces the portal with login options for students and admins.

### 🎓 Student Login (`pages/student-login.html`)
Students log in using their credentials to access assigned exams.

### 📝 Student Registration (`pages/student-register.html`)
New students can create an account to access the examination system.

### 📊 Student Dashboard (`pages/student-dashboard.html`)
After login, students can view available exams, attempt them, and see results.

### 👨‍💼 Admin Login (`pages/admin-login.html`)
Administrators log in to manage exams, questions, and student data.

### ℹ️ About (`pages/about.html`)
Information about the project and the college.
